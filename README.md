# 🎤 SpeakUp – AI English Tutor for Malayalam Speakers

## 🌟 About the Project

SpeakUp is a simple yet powerful AI-based voice assistant designed especially for Malayalam speakers who want to improve their English speaking skills.

The idea came from a common problem — many people understand English but feel shy or afraid to speak. This platform creates a safe and friendly space where users can practice speaking without fear of being judged.

Users can talk naturally in Malayalam or English, and the AI listens, corrects mistakes, and responds like a supportive teacher. It also speaks the response aloud, helping users improve both communication and pronunciation.

---

## 🚀 Features

* 🎤 Voice-based interaction (Speech Recognition)
* 🔊 AI voice response (Text-to-Speech)
* 🧠 Real-time grammar correction
* 💬 Natural conversation like a human tutor
* 🌐 Malayalam to English support
* 🎯 Topic-based practice (daily life, travel, etc.)
* 👩‍🏫 Teacher-style guidance with explanations
* 💙 Friendly and encouraging responses

---

## 🛠️ Technologies Used

* HTML, CSS, JavaScript (Frontend)
* Web Speech API (Voice Input & Output)
* Node.js + Express (Backend)
* OpenRouter API (AI Integration)
* GitHub Pages (Frontend Hosting)
* Render (Backend Hosting)

---

## ⚙️ How It Works

1. User clicks the mic and speaks
2. Speech is converted into text
3. Text is sent to the backend server
4. AI processes and corrects the sentence
5. AI responds in simple English
6. Response is spoken aloud for practice

---

## 🎯 Objective

The main goal of this project is to help people overcome hesitation in speaking English by providing a safe, interactive, and judgment-free learning environment.
